Here are some commands to install Ansible on Linux machines:

`Master Node`
1. `sudo apt-get update`
2. `sudo apt-get upgrade`
3. `sudo apt-get -y install ansible`

`Slave Node`
1. `sudo apt-get update`
2. `sudo apt-get upgrade`
3. `sudo apt-get install openssh-server`
4. `sudo systemctl start ssh`
